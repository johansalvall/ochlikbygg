# Agent Roles Overview

## Architecture Agent
- Maintains structure, components, and semantic HTML/CSS.
- Provides the blueprint for the website.
- Ensures accessibility and responsive design.

## Design Agent
- Styles the website based on the blueprint.
- Focuses on colors, fonts, spacing, and image/text placement.
- Ensures visual consistency and modern aesthetics.

### Workflow
1. Architecture Agent sets structure.
2. Design Agent applies styling within the structure.
3. Optional Content Agent (not included) can provide text/images to be styled.
